# OpenGL learning

This is a learning repository for dealing with OPENGL.

## Dependencies
- glew
- glfw3

## Install
### MAC
This assumes you have [homebrew](https://brew.sh/) installed on your device.

```bash
$ brew install glfw3
```
## Build
### MAC
```bash
$ g++ main.cpp -framework OpenGl -std=c++17 -lglfw -lglew -o demo
```

## Run
```bash
$ ./demo
```
